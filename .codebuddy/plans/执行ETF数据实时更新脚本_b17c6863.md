## 用户需求

执行实时更新ETF数据的脚本，更新ETF实时价格和配置信息到数据库。

## 产品概述

ETF-Insight是一个专业的ETF分析与对比平台，包含美股ETF和A股红利ETF的数据管理功能。

## 核心功能

1. **ETF数据同步**：更新美股ETF（QQQ、SCHD、VOO等）的实时价格数据（模拟数据）
2. **ETF配置管理**：更新ETF基本信息如名称、类别、费率等
3. **数据库集成**：将更新的数据写入SQLite数据库（etf_insight.db）
4. **实时显示**：更新后在前端展示最新的ETF价格和涨跌幅信息

## 技术栈选择

- **后端语言**: Go 1.25.0
- **数据库**: SQLite (etf_insight.db)
- **依赖管理**: Go Modules
- **构建工具**: Go build / go run

## 实现方案

### 技术架构

使用项目中现有的命令行工具 `backend/cmd/syncetf/main.go` 来执行ETF数据同步。该工具：

1. 初始化数据库连接
2. 遍历预定义的ETF列表（包含QQQ、SCHD、VNQ、VYM等常见ETF）
3. 生成模拟的实时价格数据（基于基准价格的随机波动）
4. 更新ETF配置信息到数据库
5. 保存实时交易数据到etf_data表

### 实现细节

#### 核心算法

1. **价格模拟算法**: 基于基准价格生成±1.5%范围内的开盘价和±1.0%范围内的收盘价波动
2. **成交量模拟**: 随机生成100万到5000万的成交量数据
3. **数据持久化**: 使用GORM的FirstOrCreate方法避免重复数据

#### 执行方式

项目提供多种执行方式，本方案采用最直接的Go命令行执行方式：

- 在项目根目录直接使用 `go run backend/cmd/syncetf/main.go`
- 或者进入目录使用 `cd backend/cmd/syncetf && go run main.go`

### 性能考虑

- **内存使用**: 极低，仅处理内存中的ETF列表
- **执行时间**: 预计1-2秒完成所有ETF数据更新
- **数据库优化**: 使用事务和批量操作确保数据一致性
