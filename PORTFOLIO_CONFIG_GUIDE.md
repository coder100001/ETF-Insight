# ETF组合配置管理功能使用指南

## 功能概述

ETF组合配置管理模块是一个功能完整的投资组合配置系统，支持创建、编辑、删除、分析和保存投资组合配置方案。

## 主要功能

### 1. 组合配置管理
- ✅ **创建组合**：自定义投资组合名称、描述和配置方案
- ✅ **编辑组合**：修改现有组合的配置参数
- ✅ **删除组合**：删除不需要的组合配置
- ✅ **复制组合**：基于现有组合快速创建新组合
- ✅ **状态管理**：启用或禁用组合配置

### 2. 权重配置
- ✅ **自由分配**：为每个ETF设置0-100%的权重
- ✅ **滑块调整**：直观的滑块控制权重分配
- ✅ **自动平衡**：一键自动平衡所有ETF权重
- ✅ **重置功能**：快速重置所有权重为0
- ✅ **实时验证**：自动验证权重总和是否为100%

### 3. 实时计算和分析
- ✅ **组合价值计算**：基于实时价格计算组合总价值
- ✅ **收益分析**：计算总收益、收益率、股息收入
- ✅ **风险评估**：分析组合风险指标和波动率
- ✅ **预览分析**：快速预览组合分析结果
- ✅ **完整分析**：跳转到详细分析页面

### 4. 配置管理
- ✅ **筛选功能**：按状态（全部/启用/禁用）筛选组合
- ✅ **搜索功能**：按名称或描述搜索组合
- ✅ **快速加载**：一键加载预设配置到分析页面
- ✅ **配置对比**：对比不同组合的性能指标

## 使用说明

### 创建新组合

1. **点击"创建新组合"按钮**
   - 打开组合配置创建模态框

2. **填写基本信息**
   - **组合名称**：输入组合名称，如"433组合"、"稳健组合"等
   - **组合描述**：简要描述组合的投资策略和目标

3. **设置投资金额**
   - 输入投资金额（美元），建议范围：$1,000 - $1,000,000
   - 系统会自动验证金额有效性

4. **配置权重分配**
   - 使用滑块或输入框设置每个ETF的权重
   - 权重范围：0-100%
   - 确保所有权重总和为100%
   - 可使用"自动平衡"功能快速分配

5. **保存配置**
   - 点击"保存配置"按钮
   - 系统会验证配置的有效性
   - 保存成功后自动刷新列表

### 编辑现有组合

1. **找到要编辑的组合**
2. **点击组合卡片右上角的"..."菜单**
3. **选择"编辑配置"**
4. **修改配置信息**
5. **保存更改**

### 复制组合

1. **找到要复制的组合**
2. **点击"..."菜单**
3. **选择"复制配置"**
4. **系统自动创建同名副本**
5. **编辑副本配置**

### 加载配置到分析

**方法1：快速加载**
1. **点击"..."菜单**
2. **选择"加载此配置"**
3. **自动跳转到组合分析页面**
4. **应用配置参数**

**方法2：预览后加载**
1. **点击"分析此配置"**
2. **查看预览结果**
3. **点击"查看完整分析"**
4. **跳转到详细分析页面**

### 删除组合

1. **点击"..."菜单**
2. **选择"删除配置"**
3. **确认删除操作**
4. **配置将被永久删除**

## 预设组合说明

系统预置了6个常用组合配置：

### 1. 433组合
- **SCHD**: 40%
- **SPYD**: 30%
- **JEPQ**: 30%
- **特点**：保守配置，适合稳健型投资者

### 2. 442组合
- **SCHD**: 40%
- **SPYD**: 40%
- **JEPQ**: 20%
- **特点**：平衡配置，兼顾收益与风险

### 3. 平衡型组合
- **SCHD**: 30%
- **SPYD**: 20%
- **JEPQ**: 15%
- **JEPI**: 20%
- **VYM**: 15%
- **特点**：多ETF均衡配置，分散投资风险

### 4. 稳健型组合
- **SCHD**: 40%
- **SPYD**: 20%
- **JEPQ**: 10%
- **JEPI**: 20%
- **VYM**: 10%
- **特点**：以高股息ETF为主的稳健投资策略

### 5. 激进型组合
- **SCHD**: 20%
- **SPYD**: 20%
- **JEPQ**: 60%
- **特点**：以JEPQ为主的高收益高风险配置

### 6. 纯股息组合
- **SCHD**: 35%
- **SPYD**: 35%
- **VYM**: 30%
- **特点**：专注于股息收益的组合配置

## API接口说明

### 获取所有组合配置
```
GET /workflow/api/portfolio-configs/
```

### 获取单个配置详情
```
GET /workflow/api/portfolio-configs/<config_id>/detail/
```

### 创建组合配置
```
POST /workflow/api/portfolio-configs/
Content-Type: application/json

{
  "name": "组合名称",
  "description": "组合描述",
  "total_investment": 10000,
  "allocation": {
    "SCHD": 0.4,
    "SPYD": 0.3,
    "JEPQ": 0.3
  },
  "status": 1
}
```

### 更新组合配置
```
PUT /workflow/api/portfolio-configs/<config_id>/
PATCH /workflow/api/portfolio-configs/<config_id>/
```

### 删除组合配置
```
DELETE /workflow/api/portfolio-configs/<config_id>/
```

### 切换配置状态
```
POST /workflow/api/portfolio-configs/<config_id>/toggle-status/
```

### 分析投资组合
```
POST /workflow/api/analyze-portfolio/
Content-Type: application/json

{
  "allocation": {
    "SCHD": 0.4,
    "SPYD": 0.3,
    "JEPQ": 0.3
  },
  "total_investment": 10000,
  "tax_rate": 0.10
}
```

### 获取组合性能指标
```
GET /workflow/api/portfolio-configs/<config_id>/performance/
```

### 获取统计信息
```
GET /workflow/api/portfolio-stats/
```

## 注意事项

### 权重配置
- ⚠️ 所有权重总和必须等于100%
- ⚠️ 权重范围为0-100%
- ⚠️ 至少配置一个ETF的权重
- ✅ 使用自动平衡功能确保总和正确

### 投资金额
- ⚠️ 最小投资金额为$100
- ✅ 建议投资金额：$1,000 - $1,000,000
- ✅ 可以在分析页面随时调整

### ETF选择
- ⚠️ 只能配置启用的ETF
- ⚠️ ETF必须存在于系统中
- ✅ 可以在"ETF配置"页面管理ETF

### 数据时效性
- ⚠️ 组合分析基于最新的ETF价格
- ⚠️ 价格数据可能不是实时数据
- ✅ 定期更新ETF价格数据

## 故障排除

### 权重总和错误
**问题**：提示"权重总和应为100%"
**解决**：
- 检查所有ETF权重输入
- 使用"自动平衡"功能
- 手动调整确保总和为100%

### ETF配置失败
**问题**：提示"ETF不存在或已禁用"
**解决**：
- 检查ETF是否已启用
- 在"ETF配置"页面启用ETF
- 确认ETF代码正确

### 分析失败
**问题**：组合分析出错
**解决**：
- 检查ETF价格数据是否最新
- 更新ETF实时数据
- 确认权重配置正确

### 配置加载失败
**问题**：无法加载配置到分析页面
**解决**：
- 检查配置是否启用
- 确认权重配置有效
- 刷新页面重试
