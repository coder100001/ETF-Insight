实时数据更新功能测试

功能说明

投资组合分析页面已添加实时数据更新功能，可以手动触发获取最新的ETF价格数据。

  • 一键更新：点击页面顶部的"更新实时数据"按钮
  • 即时反馈：显示加载状态和更新结果
  • 错误处理：完善的错误提示和重试机制
  • 动画效果：成功更新后有平滑的动画反馈

测试步骤

  1. 点击下方按钮跳转到投资组合分析页面
  2. 找到页面右上角的绿色"更新实时数据"按钮
  3. 点击按钮，观察加载状态提示
  4. 等待几秒钟，查看更新结果
  5. 检查页面数据是否已更新
  6. 验证所有指标（总投资、当前价值、收益率等）是否正确显示

API测试

可以使用以下代码在浏览器控制台测试API：

// 测试更新实时数据API
fetch('/workflow/api/update-realtime/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-CSRFToken': document.querySelector('[name=csrfmiddlewaretoken]').value
    },
    body: JSON.stringify({
        allocation: {
            'SCHD': 0.4,
            'SPYD': 0.3,
            'JEPQ': 0.3
        },
        total_investment: 10000
    })
})
.then(response => response.json())
.then(data => console.log(data));

注意事项

  • API限流：频繁更新可能触发限流，建议间隔至少10秒
  • 市场时间：市场开市时数据更新更频繁
  • 网络延迟：更新时间取决于网络速度和API响应
  • 缓存策略：更新会清除所有缓存，确保数据最新
